Tratamiento de datos

  1. Esta aplicación es de uso estrictamente pedagogica los datos que se capturen durante su uso seran borrados periodicametne
  2. Se mezclaran con datos de prueba y datos de control
  3. Los datos seran almacendos siguindo estricos protocolos de seguridad
  4. Las contraseñas se encriptaran y no seran visibles o accesibles desde la base de datos
  5. Se aconcejan no incluir dataos personales como contraseñas que uses en otras aplicacions o datos de contacto muy espesificos
  6. Si tienes alguna duda o inquietud sobre el tratamiento de tus datos puedes contactarnos en el siguiente enlace Contacto
